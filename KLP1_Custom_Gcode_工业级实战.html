4 个钩子位置

SuperSlicer 的 Custom G-code 在 Settings → Printer Settings → Custom G-code 中配置，分为 4 个执行时机：

📍 Settings → Printer Settings → Custom G-code
Before layer change G-code → 层切换前

在打印层切换之前执行，适合做喷嘴清理、位置调整等准备工作。

; 层切换前 - 喷嘴清理宏
; 适用于 Silk PLA 等易拉丝材料
G91              ; 相对定位
G1 Z5 F600       ; 抬升 5mm 避免碰撞
G1 X5 Y0 F3000  ; 快速移动甩丝
G1 X-5 F3000   ; 往复清理
G90              ; 恢复绝对定位
M117 层{layer_num}完成
After layer change G-code → ⭐ 最常用

层切换完成后执行，最常用的钩子位置，适合传感器触发、进度上报等。

; 层切换后 - 探针检测宏
; 每 10 层执行一次 Z 探针检测
{if layer_num % 10 == 0 && layer_num > 0}
  M117 探针_第{layer_num}层
  G91
  G1 Z5 F600
  G90
  G1 X110 Y110 F9000
  G30
  {if printer.probe.last_result > 0.05}
    M112  ; 超差急停
  {endif}
{endif}
Between objects G-code → 多对象之间

打印多个对象时，在完成一个对象后、开始下一个对象前执行。

; 多对象切换提示
; 适合长时间打印的进度提醒
{if total_layer_number > 50}
  M118 OBJECT_CHANGE Layer={layer_num}
  M300 S800 P300  ; 蜂鸣提醒
  RESPOND MSG="Object {layer_num}/{(total_layer_number/50)|int}"
{endif}
Change filament G-code → 换料时

在换料时执行，适合做喷嘴清理、颜色切换提示等。

; 换料提醒宏
; 多色打印时提醒操作员
M117 换料: {filament_name}
G91
G1 E-8 F300   ; 退料
G1 Z10 F600   ; 抬升
G90
RESPOND MSG="请插入 {filament_name}"

4 个占位符变量

SuperSlicer 支持的层相关占位符，在切片时会被替换为实际数值：

📏
{layer_num}
当前层号
🔧 触发条件判断 · 层数计数 · 进度跟踪
示例值: 0 → 1 → 2 → ... → 147
📐
{layer_z}
当前 Z 高度
📏 Z 探针对齐 · 位置校准 · 高度验证
示例值: 0.2mm → 0.4mm → ... → 29.4mm
📊
{total_layer_number}
总层数
📊 进度百分比计算 · 完成度显示 · 耗时估算
示例值: 147 (全程打印)
🏷️
{filament_name}
耗材名称
🏷️ 多料标记 · 批次追踪 · 颜色区分
示例值: PLA, PETG, ABS, TPU, Silk Gold

FAQ 常见问题解答

Custom G-code 在哪一层生效？

Custom G-code 在切片时进行变量替换，而不是打印运行时执行。SuperSlicer 在生成 G-code 时会把 {layer_num} 等占位符展开为实际数值，Klipper 收到的文件里已经没有占位符了。

{layer_num} 是从 0 还是 1 开始？

从 0 开始。第 1 层打印时 layer_num = 0，第 2 层时 layer_num = 1。
注意：layer_num == 0 表示首层，layer_num == 1 表示第 2 层。

RESPOND MSG 在 Mainsail 哪里看？

在 Mainsail 或 Fluidd 的 Console 标签页。Moonraker 会将 RESPOND 消息通过 WebSocket 转发，实时显示在控制台。也可在 OctoPrint 的 Terminal 标签查看。

M117 和 M118 区别？

M117 = LCD 显示（如果有屏幕则显示文字）；M118 = 通过 Moonraker 转发到 WebSocket，可被 OctoPrint/Fluidd/Mainsail 捕获。M118 更适合远程监控场景。

SuperSlicer 切片时变量替换还是 Klipper 运行时替换？

切片时替换。 SuperSlicer 在生成 G-code 时会计算 {if layer_num % 10 == 0} 等条件表达式，并把结果写入输出文件。Klipper 只是执行已经展开的纯 G-code。

{if ... } 表达式支持函数吗？

支持。常用数学函数：round()、floor()、ceil()、abs()。
示例：{(layer_num / 10)|round} 计算层数除以 10 的四舍五入值。

Klipper 能用 Marlin 的 M 代码吗？

约 80% 兼容。常用代码（M112、M114、M117、M118、G4、M300）都兼容。温度相关代码建议用 Klipper 原生命令。

探针急停阈值 0.05mm 怎么来的？

基于 BLTouch 的重复精度 ±0.05mm（官方规格）。超过此值可能表示：热床未调平、Z offset 偏移、材料膨胀/收缩、喷嘴堵塞。需要人工检查。

FBG 光纤监测需要什么硬件？

需要：① FBG 传感器解调仪（如 Micron Optics sm130、Bristol mIRR）；② 光纤光栅传感器埋在打印件内部；③ 通过 M118 触发外部设备采样记录。

CT 标记微珠用什么材料？

通常用钨粉（W）或铅玻璃珠，X 射线对比度高，直径 0.2-0.5mm。用量极小（每点约 0.015mm³），不影响打印件结构强度，便于 CT 配准定位。
