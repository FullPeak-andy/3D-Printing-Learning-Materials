1. 机械装配类

机架水平、皮带张力、导轨垂直度、热床平整度

🔲 机架水平校准

确保打印机底座处于水平状态，这是所有后续校准的基准。使用水平仪在 X/Y/Z 三个方向检测。

检查标准
项目标准值工具
XY 平面水平度< 0.5mm/m水平仪
对角线偏差< 0.5mm卷尺
框架扭曲无晃动手动检查

⚡ XY 皮带张力

CoreXY 皮带的张力直接影响打印精度。两侧皮带张力必须相等，否则会导致 gantry 倾斜。

⚠️
皮带张力不均会导致斜向 ringing 纹和椭圆形打印问题。
推荐张力值
机型类型频率范围张力(估算)
Voron V0110-130 Hz~8-10 N
Voron V2.4/Trident80-100 Hz~6-8 N
Bambu X1/P160-75 Hz~5-7 N
KINGROON KLP190-110 Hz~7-9 N

📏 Z 导轨垂直度

Z轴两根导轨必须完全平行且垂直于 XY 平面。对于双 Z 电机机型，两侧高度必须一致。

CoreXY Z轴校准
步骤方法
1. 手动检测移动 gantry 到中间位置，用直角尺检测
2. 电机独立调节松开联轴器，分别调整每根导轨
3. 运行 QUAD_GANTRY_LEVELKlipper 自动补偿（V2.4）
4. 运行 Z_TILT_ADJUSTKlipper 自动调平（Trident）

🛏️ 热床平整度

热床本身和安装方式都可能导致变形。建议加热后测量，模拟实际打印状态。

热床检测标准
状态平整度处理方式
冷态< 0.1mm直接检测
热态(60°C)< 0.2mm加热后检测
极端变形> 0.5mm更换或加垫片

🔩 皮带轮/惰轮紧固

检查所有皮带轮、惰轮、联轴器的固定螺丝。振动可能导致螺丝松动。

# 紧固检查清单：GT2 皮带轮螺丝、惰轮轴承座、电机固定螺丝、联轴器顶丝

🔌 线缆整理

确保所有线缆走线合理，不影响运动部件。使用线缆夹和束线带固定。

💡
CoreXY 运动复杂，线缆必须留有足够余量并固定在非移动部件上。

✅ 机械装配 Checklist

  • 检查机架所有连接螺丝是否紧固（2020型材角码连接）
  • 测量框架对角线，确保误差 < 0.5mm
  • 检测 XY 平面水平度（水平仪）
  • 安装并检查皮带张力（频率计或 pluck test）
  • 确认两侧皮带张力相等（CoreXY 关键）
  • 检查 Z 轴导轨平行度和垂直度
  • 紧固所有皮带轮/惰轮螺丝
  • 检查联轴器是否居中、有无松动
  • 检测热床平整度（冷态/热态）
  • 整理线缆，确保不干涉运动部件
  • 手动移动各轴，确认无卡顿/异响
  • 涂抹润滑脂到 Z 丝杠（如果适用）

2. 电气/电子类

主板固件、TMC驱动配置、热端/热床接线、风扇、限位开关

🔧 主板固件刷写

KINGROON KLP1 预装了 Klipper 固件，但对于新主板或需要更新时需要重新刷写。

Klipper 固件编译步骤
# 1. 在树莓派/主机上安装 Klipper
git clone https://github.com/Klipper3d/klipper
cd klipper

# 2. 配置编译选项
make menuconfig
# 选择: 
#   Processor architecture: ARM (STM32)
#   Processor model: STM32F103
#   Build options:
#     [] Use USB for communication (需要CAN或USB)
#     [x] Use serial for communication
#     Serial port: /dev/ttyAMA0 或 /dev/serial/by-id/xxx

# 3. 编译固件
make

# 4. 将 klipper.bin 复制到 SD 卡
# 重命名为 firmware.bin 并刷入
💡
KINGROON KLP1 出厂已预装 Fluidd/Mainsail 界面，首次调试可直接在 Web 界面操作。

✅ 电气配置 Checklist

  • 确认主板固件版本（Klipper/Fluidd/Mainsail）
  • 检查所有电机接线相序（X/Y/Z/E）
  • 配置 TMC2209 run_current（电机额定×80%）
  • 设置 microsteps 为 16（或 32）
  • 验证热端/热床加热器接线牢固
  • 测试热敏电阻读数（室温 ~100kΩ）
  • 检查风扇接线方向和 PWM 控制
  • 测试 X/Y/Z 限位开关触发
  • 验证探针（BLTouch）部署和触发
  • 检查电源电压（24V 或 12V）匹配
  • 确认接地良好

3. 软件配置类

printer.cfg 核心配置段详解

📄 printer.cfg 基础配置

KINGROON KLP1 使用 Klipper，配置文件是 printer.cfg。以下是核心配置段的详细说明。

[stepper_x] / [stepper_y] / [stepper_z] 步进电机配置

▼ 
[stepper_x]
step_pin: PF13
dir_pin: PF12
enable_pin: !PF14
microsteps: 16
rotation_distance: 40    # 皮带驱动计算值

# 对于 GT2 皮带 + 20T 皮带轮:
# rotation_distance = 齿数 × 齿距 = 20 × 2 = 40mm

# 或者使用 steps_per_mm 转换:
# rotation_distance = full_steps_per_rotation × microsteps / steps_per_mm
# = 200 × 16 / 80 = 40

endstop_pin: ^PG6        # X 轴限位开关
position_min: 0
position_max: 220        # 根据机型调整
position_endstop: 0

# CoreXY 需要特殊运动学
[stepper_x]
kinematics: corexy
max_z_velocity: 50
max_z_accel: 1000

🎸 皮带张力校准

使用频率计或手机频谱分析 app 测量皮带振动频率。

$ 频率目标：CoreXY A/B 皮带均约 110 Hz（150mm 跨度）
💡
手机 app 推荐：Spectroid（Android）、Decibel X（iOS）。测试时 gantry 位于皮带中间位置。

📐 Z 导轨垂直度校准

确保 Z 轴两根导轨平行且垂直于 XY 平面。

# G28 # 归零 Z_TILT_ADJUST # Trident 双 Z 自动调平 # 或 QUAD_GANTRY_LEVEL # V2.4 四点 Z 调平

🔧 SCREWS_TILT_CALCULATE 热床调平

使用探针测量各调平螺丝位置的 Z 高度，计算调整量。

# G28 SCREWS_TILT_CALCULATE

输出示例：
front left screw : adjust 00:00 (基准)
front right screw : adjust CW 00:02 (顺时针转2圈)
rear right screw : adjust CCW 00:01 (逆时针转1圈)

📡 PROBE_CALIBRATE 探针 Z-offset

校准探针触发点与喷嘴末端的距离差。

# G28 PROBE_CALIBRATE

操作流程：
1. 探针移动到热床中心
2. 使用 A4 纸进行"纸测"
3. ACCEPT 确认
4. SAVE_CONFIG 保存

📊 bed_mesh 网格测量

创建热床高度补偿网格，用于补偿热床不平整。

# BED_MESH_CALIBRATE
💡
KINGROON KLP1 建议 6×6=36 点网格。打印大件时效果明显。

📏 e-steps / rotation_distance 挤出校准

确保 100mm 挤出命令实际挤出 100mm 耗材。

# G91 G1 E100 F100 G1 E-100 F100 # 测量实际挤出量并计算修正值

修正公式：
New rotation_distance = Old × (期望值 / 实测值)

消除拐角过冲和欠冲，提升打印精度。

📦 XYZ Cube 校准立方

最基础的测试打印，用于检验尺寸精度和整体打印质量。

推荐参数
参数推荐值
尺寸20mm × 20mm × 20mm
层高0.2mm
壁数3（0.8mm）
填充15-20%

🌡️ 温度塔 (Temperature Tower)

测试不同温度下的打印质量，找到最佳打印温度。

# OrcaSlicer / PrusaSlicer 内置温度塔 或手动 G-code: # 每 10mm 降低 5°C # 测试范围：190°C - 230°C (PLA)

观察要点：
• 拐角清晰度
• 底部附着力
• 层间结合力
• 表面光洁度

↩️ 回抽塔 (Retraction Tower)

测试不同回抽距离/速度下的挂丝情况。

# 建议起始值： 回抽距离：4-6mm（直接驱动） 回抽速度：25-45mm/s 测试步进：每层变化 0.2mm 或 5mm/s

Kingroon KLP1 推荐：回抽 3-5mm，速度 30-40mm/s

💧 流量塔 (Flow Rate Tower)

测试不同流量值下的挤出均匀性。

# OrcaSlicer 内置 Flow Rate 校准 测试范围：85% - 105% 步进：2% 目标：找到最均匀壁厚的流量值

🔲 首层测试 (First Layer Test)

首层质量是所有打印成败的关键，必须细致调整。

首层诊断
现象原因调整方法
线条分离有间隙Z 偏高减小 Z offset / 降低喷嘴高度
线条被压平有隆起Z 偏低增加 Z offset / 升高喷嘴高度
