10 大核心特性

🏔️
Dynamic Layer Height (DLH)
⭐⭐⭐⭐⭐
模型曲面斜率自动调层高，平坦区粗、陡峭区细。精度与速度完美平衡。
普通层高 0.2mm
DLH 0.15-0.28mm 自适应
曲面斜率 45°
# 启用 DLH Print Settings → Quality → Adaptive layers # 标记斜率阈值 слой = 0.2 # 基准层高 mark = 5 # 平坦区 mark = 10 # 陡峭区
齿轮组精度±0.12mm → ±0.07mm
微缩龙 Ra0.92 → 0.71μm
悬臂桥0.3mm → 0mm
🎯
PolyHole 圆孔杀手锏
⭐⭐⭐⭐⭐
圆孔扫描时起点和终点对齐，避免普通 slicer 的圆孔接缝偏差。
普通切片
PolyHole
孔径调节 15mm
# 启用 PolyHole Print Settings → Advanced → Seam position → Aligned Print Settings → Quality → Use "PolyHole" # 应用：紧配合轴承/螺栓孔 # 圆度可达 ±0.05mm
🪜
3D Interface for Supports
⭐⭐⭐⭐
支撑接触面用 3D 网格而不是单层，避免支撑痕。
普通支撑
3D Interface
# 启用 3D Interface Print Settings → Support material → 3D Interface # 层数建议 3-5 层 support_interface_layers = 3

Custom G-code per Layer
⭐⭐⭐⭐⭐
4 个钩子 + 4 变量 + 3 套工业级宏，实现层级别 G-code 控制。
# SuperSlicer 特有钩子 before_layer_gcode # 层开始前 after_layer_gcode # 层结束后 layer_gcode # 层中任意位置 color_change_gcode # 换色时 # 可用变量 [layer_num] # 当前层号 [layer_z] # 当前 Z 高度 [current_x] # X 坐标 [current_y] # Y 坐标

⭐⭐⭐⭐

厂商预设（Vendor Bundles）

第三方厂商打包分发打印机/耗材/打印 preset，开箱即用。

Creality
Andy K1 Max 必装
包含 Ender/K1 系列打印机预设，支持官方耗材 profile。
安装步骤
1. Preferences → Configuration → Vendor
2. 勾选 Creality
3. 选择版本安装
Caribou
Voron 2.4 官方推荐
专为 Voron 系列优化，Caribou 0.1.7-susi3 是 SuperSlicer 的最佳 Voron 预设。
安装步骤
1. 下载 Caribou bundle
2. File → Import Config Bundle
3. 选择 voron_2.4.ini
KINGROON
KLP1 不在列表
⚠️ KINGROON 不在 SuperSlicer 官方 vendor 列表，KLP1 需要手搓配置。
解决方案
1. 使用 PrusaSlicer 的 KLP1 导出
2. 或参考 #57 KLP1 切换指南
3. 手动配置 printer.cfg
BIQU
Eddy + Octopus
包含 BX/HX 系列预设，Eddy 探针配置 + Octopus 主板预设。
安装步骤
1. Preferences → Vendor
2. 勾选 BIQU
3. 加载 BIQ_Octopus.ini

KLP1 切换 4 步指南

1
安装 SuperSlicer + 关闭 Wizard
下载 2.5.59.13 稳定版，安装后首次运行不要跑向导
# 安装后不要点击 "Run Configuration Wizard" # 直接关闭或最小化 # 等第 3 步注入 G-code 后再配置
2
Add Printer → Klipper → 220×220mm
选择 Klipper 驱动，设置 KLP1 的床面尺寸
# Printer Settings Printer → Klipper Bed Shape → 220 × 220 mm Use relative E codes → 是
3
Start G-code 注入 KLP1 配置
包含 PID 调谐、Z 探针校准、ABL 补偿的完整启动脚本
# Start G-code M104 S{first_layer_temperature[0]} ; 目标温度 M109 S{first_layer_temperature[0]} ; 等待 G28 ; 归零 PID_TUNE ; PID 调谐 PROBE ; Z 探针校准 G29 ; ABL 补偿
4
Difficulty 切 Advanced
暴露 80% 工业参数，解锁 Calibration 菜单
# 菜单路径 Help → Switch to... → Advanced # 解锁功能 # ✓ Calibration 完整菜单 # ✓ Dynamic Layer Height # ✓ PolyHole # ✓ 3D Interface # ✓ Custom G-code per Layer

5 个警告

👤
supermerill 单人维护
SuperSlicer 由 supermerill 一个人维护，更新节奏不稳定。开发进度依赖个人时间投入。
🔒
2.5.59.13 是最后稳定版
2.5.59.13 是目前最后一个稳定版本，supermerill 已宣布 2.5.x 分支进入维护模式。
⚠️
2.7 Beta 不上生产
2.7.x Beta 含 G2/G3 弧线拟合等新特性，但稳定性未知。测试可用，生产环境勿用。
🍎
Mac Intel 必须 xattr -cr
macOS High Sierra 以上版本，Intel Mac 安装后必须执行 xattr -cr /Applications/SuperSlicer.app。
🚫
K1 Max 走 OrcaSlicer 2.2.x
K1 Max + Creality OS 有 LAN mode crash 问题。建议使用 OrcaSlicer 2.2.x + PrusaSlicer 混合方案。

FAQ

Q: SuperSlicer 主开发者是谁？
supermerill，单人维护。GitHub: github.com/supermerill
Q: 2.5.x 还更新吗？
长期维护。2.5.59.13 是最后稳定版，2.5.x 分支会持续修复 bug，但不会增加新特性。
Q: 2.7.x 是下一代吗？
是的。2.7.x 含 G2/G3 弧线拟合、6 层对齐接缝、强制悬空速度、Klipper EXCLUDE_OBJECT 等新特性。
Q: Voron 2.4 装哪个 preset？
Caribou 0.1.7-susi3 是 SuperSlicer 的最佳 Voron 预设。从 Vendor Bundle 或 GitHub 下载。
Q: KLP1 厂商预设在哪？
⚠️ KINGROON 不在 SuperSlicer 官方 vendor 列表。KLP1 需要手搓配置，参考 #57 KLP1 切换 4 步指南。
Q: Custom G-code 在哪？
请参考教程 #56《Custom G-code per Layer 详细宏》，里面有完整的 4 钩子 + 4 变量 + 3 套工业级宏。
Q: Difficulty 切 Advanced 后还能切回吗？
可以。菜单 Help → Switch to... → Simple 或 Beginner 即可切回。参数不会丢失，只是隐藏。
Q: SuperSlicer 支持 Klipper 吗？
是，SuperSlicer 是 Klipper 一等公民。支持 [exclude_object]、宏变量、M600 换色等 Klipper 特有功能。
Q: SuperSlicer 兼容 OrcaSlicer 配置文件吗？
基本兼容，但部分参数不通用（如 AMS 相关配置）。建议独立配置以避免意外行为。
Q: 拓竹 A1 能不能用 SuperSlicer？
能用，但 AMS lite 走 OrcaSlicer 更好。SuperSlicer 不支持 AMS lite 的完整功能。
