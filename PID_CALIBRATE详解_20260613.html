🧪 PID_CALIBRATE 命令详解

Klipper 自动整定 PID 参数 · 5 分钟搞定

📋 1. 命令格式

PID_CALIBRATE HEATER=<config_name> TARGET=<temp> [WRITE_FILE=1]
参数必选说明常用值
HEATER 必填 要校准的加热器段名（[heater_*] 段） extruder（热端）/ heater_bed（热床）
TARGET 必填 目标温度（°C） 热端 220 / 热床 60
WRITE_FILE 可选 Klipper 0.12+：写临时文件供 SAVE_CONFIG 读取 默认 1（推荐）

💡 KLP1 实际命令（你刚才发的）：

PID_CALIBRATE HEATER=extruder TARGET=220

→ 校准热端 PID，目标温度 220°C（PLA/PETG 标准挤出温度）

⚙️ 2. PID 三要素（拖滑块看曲线）

温度控制 = P(比例) + I(积分) + D(微分)

20.0
4.0
40.0

🔵 蓝线 = 实际温度　⚪ 灰虚线 = 目标 220°C　横轴 = 时间(秒)

参数作用调大后果调小后果
Kp 比例 温度差多大 → 加热多猛 升温快、震荡大 升温慢、稳态误差
Ki 积分 累积误差 → 消除偏差 消除误差慢、震荡 温度上不去
Kd 微分 预测趋势 → 阻尼过冲 响应迟钝 过冲到 230°C

🔄 3. 校准流程（约 5 分钟）

  1. 预热：Klipper 加热到 TARGET（220°C）
  2. 稳态：目标温度 ±2°C 保持 30 秒
  3. 震荡测试：自动跑 3 个震荡周期（加热到 230°C → 断电到 210°C）
  4. 计算：Ziegler-Nichols 法算 Kp / Ki / Kd
  5. 输出：控制台显示 // pid_Kp=39.409 pid_Ki=9.383 pid_Kd=41.379
  6. ⚠️ 不自动保存 → 要手动跑 SAVE_CONFIG

📊 4. 校准前后温度对比

🔴 校准前：±5°C 震荡　🟢 校准后：±0.5°C 稳定　⚪ 目标 220°C

🔧 5. 何时需要校准

场景必要性原因
第一次装机 必跑 工厂默认 PID 适配不了所有机器
换热端 / 加热棒 / 热敏 必跑 加热功率/热敏响应变了
换主板 / 温控板 必跑 PWM 频率 / 死区可能不同
改 [extruder] 段参数 建议跑 挤出口径变了影响热负荷
换耗材（PLA→ABS） 建议跑 ABS 需要更高热端/热床温度
每次打印前 不跑 校准要 5 分钟常加热

⚠️ 6. 不校准的后果（这就是你之前的问题）

现象原因
温度震荡 ±5°C PID 参数错 → 输出忽大忽小
挤出量不稳定 温度抖 → 耗材热胀冷缩 → 线宽不均
首层拉丝/缺料 KLP1 当前症状 底行缺料 / 中行拉丝 / 顶行正常（之前看过的 3×3）
堵喷嘴（高温耗材） ABS/ASA 打印时温度掉 → 耗材固化堵头
热床温度上不去 热床 PID 错 → 加热慢/温度低 → 翘边
