🎯 调平金字塔 — 4 层从下到上

核心论点（来自 B 站视频）：没有 L1 物理调平，L4 Mesh 补偿就是给漏水的船刷漆。

L1 · 机械调平（地基，最重要）

做什么：拧 4 角弹簧螺丝，让 4 个角的 Z 高度一致

命令：SCREWS_TILT_CALCULATE

目标：4 角 Z 差 ≤ 5 丝 (0.05mm)

怎么拧：按报告"顺时针 / 逆时针 + 圈数 + 分数"拧

为什么重要：L2 L3 L4 都是软补偿，补偿不了机械偏差超过 0.1mm 的床。比如 L4 Mesh 测出来 X 方向差 2.67mm——软补偿的喷嘴要在 182mm 行程上补偿 2.67mm 高度差，每 1mm 高度差喷嘴会"摇头"——打印质量断崖下降。

L2 · 探针校准（Z 高度基准）

做什么：用 A4 纸测试喷嘴到床的距离，确定 z_offset

命令：PROBE_CALIBRATE

目标：z_offset 准确到 0.01mm，A4 纸能感受到"轻轻摩擦"

顺序：在 L1 物理调平之后做

KLP1 现状：z_offset = 1.050mm（SAVE_CONFIG 自动生成）

L3 · Z 等高（双 Z 专用，KLP1 跳过）

做什么：让 2 个 Z 电机的高度一致（双 Z 同步带同步也可以跳过）

命令：Z_TILT_ADJUST

目标：2 个 Z 电机高度差 ≤ 0.05mm

KLP1 现状：单 Z 同步带 + 双丝杠，用不到这个命令

L4 · Mesh 补偿（喷嘴跟随床型）

做什么：把床分成 5×5 或 6×6 网格，测每个点的 Z 高度，存为 Mesh

命令：BED_MESH_CALIBRATE

目标：网格 Z 范围 ≤ ±0.2mm

顺序：L1 + L2 完成之后做

KLP1 现状：缺 [bed_mesh] section，之前只测了 2 行 12 个点（应 6×6=36 个点）

🚑 12 种异常排错清单

这一 Tab 是 Andy 哥的"故障字典"——遇到问题 Ctrl+F 搜关键词。

① Z 轴归零不了

5 种表现 + 解决：

表现根因解决
一直往下落不触发Z 限位线断了 / BLTouch 没弹针查线 + 手动弹针
一直往上跑不下降G28 Z 找触发点失败手动 G1 Z10 让喷嘴下降再 G28
归零后撞床（嘎嘎声）position_min 配错 / z_offset 偏改 printer.cfg + 重测 z_offset
归零后离床 2-3mm 太远z_offset 偏大PROBE_CALIBRATE 重测
归零后离床 < 0.1mm 几乎贴z_offset 偏小PROBE_CALIBRATE 重测
② SCREWS_TILT_CALCULATE 报"section not configured"

根因：printer.cfg 缺 [screws_tilt_adjust] section

解决：在 printer.cfg 末尾追加 section（4 颗螺丝坐标）

③ BED_MESH_CALIBRATE 只测 2 行 12 个点（应 6×6=36）

根因：缺 [bed_mesh] section，Klipper 用默认的 mesh_min/max = 整张床，导致 BLTouch 行程外

解决：加 [bed_mesh] section，明确 mesh_min: 20,20 + mesh_max: 202,190 + probe_count: 6,6

④ Mainsail 工具下拉里没有"调平"按钮

根因：工具下拉只显示 [gcode_macro] 定义的宏，不是 Klipper 全部命令

解决：在控制台直接输入命令（如 BED_MESH_CALIBRATE 回车）

⑤ Klipper 状态显示 Disconnect

2 种情况：

A. SAVE_CONFIG 触发的自动重启（正常！）→ 等 10-30 秒自动重连

B. 真的挂了（异常）→ SSH sudo systemctl restart klipper

⑥ Mesh 测出来 Z 差 > 1mm 严重倾斜

根因：机械结构没调平（SCREWS_TILT_CALCULATE 没跑 / 跑没拧）

解决：先 SCREWS_TILT_CALCULATE 拧 4 角螺丝，再 BED_MESH_CALIBRATE

⑦ 第一层粘不住 / 太薄 / 太厚

粘不住：床温太低 / 喷嘴离床太远 / Z 初始高度不对

太薄（透光）：z_offset 偏大 / 喷嘴离床太近

太厚（糊）：z_offset 偏小 / 喷嘴离床太远

解决：PROBE_CALIBRATE 重测 + 切片里调 Z 高度微调

⑧ 共振补偿失败 / 报告空白

根因：ADXL345 没接 / 接线错 / [adxl345] section 错

解决：检查 MKS_THR:gpio13/14/15 接线 + 跑 G40 重新校准

⑨ PID 校准温度波动大

根因：风扇吹 / 室温不稳 / 热敏电阻接触不良

解决：关门校准 + 室温 25℃ + 检查热敏电阻

⑩ 探针不触发 / 一直触发

不触发：BLTouch 红色灯不闪 / 探针针没伸出

一直触发：BLTouch 黄灯常亮 / 探针卡住

解决：重启 BLTouch（断电 5 秒再上电）+ 检查线序

⑪ SAVE_CONFIG 块被覆盖

根因：直接编辑了 #*# 开头的自动生成块

解决：改前面 [bed_mesh] section（不带 #*#）→ SAVE_CONFIG 会更新 #*# 块

⑫ 重复 section 报错

症状：klippy.log 报 "Duplicate section [probe]"

解决：合并两个 section 删一个

📋 6 个必跑命令详解

1️⃣ G28 — 归位

作用：X/Y/Z 全部回到限位位置（机械原点）

参数：无

使用：控制台输入 G28 回车

KLP1 特殊：X/Y 用 sensorless homing（无接触限位），Z 用 BLTouch 探针

2️⃣ PROBE_CALIBRATE — 探针校准

作用：用 A4 纸测试喷嘴到床距离，确定 z_offset

使用：控制台输入 PROBE_CALIBRATE 回车

流程：

  1. 喷嘴移到床中心
  2. Z 下降到探针附近
  3. 按 TESTZ Z=-0.1 慢慢下降
  4. A4 纸感到"轻轻摩擦"时输入 ACCEPT

保存：SAVE_CONFIG

3️⃣ SCREWS_TILT_CALCULATE — 4 角螺丝校准

作用：测 4 角 Z 高度差，报告拧螺丝方向 + 圈数

使用：控制台输入 SCREWS_TILT_CALCULATE 回车

注意：需要 [screws_tilt_adjust] section

拧螺丝流程：断电 → 拧 → 上电 → 重跑

4️⃣ BED_MESH_CALIBRATE — 床 Mesh

作用：把床分成网格，测每个点 Z 高度，存为 Mesh

使用：控制台输入 BED_MESH_CALIBRATE 回车

注意：需要 [bed_mesh] section

完成：自动 SAVE_CONFIG

5️⃣ SHAPER_CALIBRATE — 共振补偿

作用：用 ADXL345 测 X/Y 轴共振频率，自动算 input shaper

使用：控制台输入 SHAPER_CALIBRATE 回车

需要：[adxl345] section + ADXL345 硬件

KLP1 现状：已有 MKS_THR:gpio13/14/15 接线，已配

6️⃣ PID_CALIBRATE — 温度 PID

作用：自动测挤出机和热床的 PID 参数

使用：

PID_CALIBRATE HEATER=extruder TARGET=245
PID_CALIBRATE HEATER=heater_bed TARGET=70

完成：报告 Kp/Ki/Kd，复制到 printer.cfg 对应 section

第一层必须 100% 完美（否则灯罩"漏光"）
