## 🚀 使用方法

**所有文件都是单文件 HTML，零依赖**：
1. 点击任意 `.html` 文件
2. 在 GitHub 页面上点 **"Download Raw File"** 或 **"View Raw"**
3. 保存到本地后双击用浏览器打开
4. 不需要联网，不需要安装任何依赖

或者直接 clone 整个仓库：
```bash
git clone https://github.com/FullPeak-andy/3D-Printing-Learning-Materials.git
cd 3D-Printing-Learning-Materials
# 双击任意 .html 在浏览器打开
```
