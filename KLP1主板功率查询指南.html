⚠️ 关键问题：300W 总额定 ≠ 加热棒能用 300W！
实际单路加热棒最大功率 取决于主板 MOSFET 型号 + 走线宽度 + 散热。
官方没明确写"加热棒最大支持多少W"，所以要自己查。
方法 1：SSH 终端查询（推荐） 最准 · 3 分钟
1
SSH 连 KLP1
电脑打开终端/PowerShell，输入：
ssh mks@你的KLP1的IP地址# 例如 ssh mks@192.168.1.100 # 密码：makerbase（默认）
不知道 IP？在 KLP1 屏幕菜单里找 "网络" → "WiFi 信息" / "有线 IP"
2
查看 Klipper 配置（最直接）
cat ~/printer_data/config/printer.cfg | grep -A 3 "extruder\]" # 输出类似： [extruder] heater_pin: PA1 max_temp: 260 min_temp: 0 sensor_type: EPCOS 100K B57560G104F
关键看 max_temp: 260 ← 主板软件锁的最高温度
3
查看 MCU 型号（间接知道功率上限）
cat /proc/cpuinfo | grep "model name" # 或更精准： dmesg | grep -i "mcu\|gd32\|stm32" # 应该能看到： MCU: GD32F303 (GD32F303VCT6) Clock: 72 MHz
GD32F303 = STM32F103 的国产替代品，IO 口驱动能力 25mA 单 IO
4
查看加热棒实际功率（动态测试）
# 在 Klipper 终端执行（mainsail/fluidd 网页 console 框）： SET_HEATER_PWM HEATER=extruder VALUE=1.0 # 100% 满功率加热 5 秒，观察温度上升速度 # 或看电源电流（如果有钳形表）： # 钳住 24V 主电源线，看电流 # 50W 加热棒 = 50/24 ≈ 2.08A # 60W 加热棒 = 60/24 ≈ 2.5A # 100W 加热棒 = 100/24 ≈ 4.17A（KLP1 危险）
💡 SSH 终极查询：导出完整 Klipper 状态
RESTART # 然后立刻看 klippy.log： tail -f ~/printer_data/logs/klippy.log # 启动时会有这一行： MCU 'mcu' config: GD32F303VCT6 Heap: 23456/32768 bytes free Clock: 72MHz
方法 2：Mainsail / Fluidd 网页界面查询 最简单 · 1 分钟
1
打开 KLP1 网页
浏览器输入：http://你的KLP1的IP地址
默认进 Mainsail 界面
2
查看机器信息
Mainsail 顶部齿轮 ⚙️ → "Dashboard" 或 "Machine"
能看到：
  • MCU 型号：GD32F303VCT6
  • Klipper 版本
  • 运行时间
  • 主机名：mks
3
查看温度传感器（动态看实际温度）
Mainsail 主界面 → "Temperature" 区域
能实时看到：
  • 🔴 Extruder：热端实际温度 + 目标温度
  • 🔵 Bed：热床实际温度 + 目标温度
  • PWM 百分比（加热棒当前占空比）
4
查看功耗（如果有智能插座）
最简单方法：给 KLP1 接一个智能插座（小米/涂鸦都行，¥30-50）
→ App 能看实时功率 W、累计 kWh、电流 A
→ 加热到 240°C 满载时，整机功率约 150-180W（加热棒 50W + 热床 100W + 主板 + 风扇等）
方法 3：拆机看主板硬件（进阶） 最权威 · 需要拆机
⚠️ 拆机风险：断电 5 分钟后再操作，避免静电。
不建议新手做——除非你已经确定要 DIY 改电路。
1
拆 KLP1 底板 / 侧板
KLP1 主板位置：底部仓（掀开底板）
需要的工具：2.5mm 内六角扳手（拆底板 4 颗螺丝）
⚠️ 先断电！拔 24V 电源线 + 等 5 分钟（电容放电）
2
找加热棒输出 MOSFET
看主板上靠近 "HE" 端口（加热棒输出）的位置
应该有一个 8-Pin SOIC 或 DPAK 封装的 MOS 管
常见型号：
  • HY2302 / SI2302（N 沟道，2A/20V，太小）
  • IRF7507（双 N 沟道，2.4A/20V）
  • AO3400A / AO3401A（5.7A/30V，够用）
  • SR340 / SS34（肖特基，3A/40V，整流用）
🔍 找到型号后：拍照 + Google "[型号] datasheet" → 看 Id（连续漏极电流）
3
查 datasheet 计算最大功率
公式：
P_max = V × I_d × 0.7 # 0.7 是降额系数（安全余量） # 例 1：AO3401A（5.7A） P_max = 24V × 5.7A × 0.7 = 95.7W # 例 2：IRF7507（2.4A） P_max = 24V × 2.4A × 0.7 = 40.3W # ← 太低了，KLP1 不可能用这个 # 例 3：双 MOS 并联（2 × IRF7507） P_max = 24V × 4.8A × 0.7 = 80.6W
💡 Andy 实际能查到的：KLP1 主板很可能是 双 MOS 并联方案（80W 安全），但厂家没写出来。
如果查到是单 MOS（40W）：那 KLP1 比想象中还弱，连 50W 都接近极限。
🎯 最简单的"猜"法：
  • 官方标配 50W 加热棒 → 主板至少支持 50W
  • Kingroon 是国产贴牌方案 → 大概率用 双 AO3401A 并联（~95W 理论 / 60W 安全）
  • 加热床输出（约 200W）→ 主板必然能扛 200W+，所以加热棒 MOS 比加热床 MOS 小
  • 结论：加热棒单路实际安全上限 60-80W（不超 MOS 规格的 70%）

🎯 你的升级结论

Andy 实际可用的加热棒范围：
  • ✅ 50W 标配（完全安全）
  • ✅ 60W 升级（主流高流量热端标准，KLP1 撑得住）
  • ⚠️ 80W 边界（理论安全但要监控 MOS 温度，建议加散热片）
