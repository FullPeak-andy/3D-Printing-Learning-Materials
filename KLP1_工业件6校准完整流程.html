First Layer 首层校准

找 Z offset（喷嘴与热床最优距离），确保首层完全贴合且表面光滑

难度 ⭐⭐⭐
耗时 30 分钟
KLP1 基线 z_offset = 1.050
1
切片操作
🖱️

点击 SuperSlicer 顶部菜单

校准菜单 → First layer calibration

SuperSlicer 会自动加载首层校准件（5×5cm 方形，5 个 Z 高度变体）

2
打印参数
参数值
层高0.2mm
打印速度30mm/s
热床温度60°C
喷嘴温度215°C
5 段 Z 高度基线 ±0.02mm
3
判定方法

观察打印件：哪一层完全贴合热床且表面光滑？

📏

理想首层：线与线之间紧密排列，边缘清晰，无气泡无间隙

← 过低：挤出的线被压扁 | 过高：线不贴合 →

4
调参公式
新 z_offset = 旧 z_offset ± 修正量
例如：测得第 3 段（z_offset + 0.02）效果最好
则新 z_offset = 1.050 + 0.02 = 1.070

把最佳值写回 printer.cfg 的 [bltouch] z_offset

Flow Rate 流量校准

找 extrusion_multiplier（实际流量），确保壁厚精确到 0.4mm

难度 ⭐⭐⭐
耗时 20 分钟
KLP1 基线 理论 1.0 / 实际 0.95-1.02
1
切片操作
🖱️

校准菜单 → Flow rate

打印 20×20×20mm 空心立方体（单壁 0.4mm）

2
打印参数
参数值
层高0.2mm
壁厚0.4mm (单壁)
速度30mm/s
填充0%
温度215°C
3
判定方法

用卡尺测量壁厚 = 实际值

测量值判断调整方向
> 0.4mm流量过高降低流量
= 0.4mm完美无需调整
< 0.4mm流量过低提高流量
4
调参公式
extrusion_multiplier = 0.4 ÷ 实测壁厚
量得 0.42mm → 0.4 ÷ 0.42 = 0.95 (流量过高)
量得 0.38mm → 0.4 ÷ 0.38 = 1.05 (流量过低)

把计算值写回 filament settings → Flow → extrusion_multiplier

❓ 常见问题 FAQ

点击展开查看答案

Q1 校准一定要按顺序做吗？ ▼
建议按顺序做，因为后续校准依赖前面的基础：

① First Layer 是所有校准的前提（Z 轴不准其他都白搭）
② 温度塔影响流量和 PA 的表现
③ 流量校准影响 PA 的准确性
④ PA 和回抽相互影响
⑤ 体积挤出率是速度上限

如果时间紧迫，First Layer + Flow Rate 是必做的核心两项。
Q2 工业件每卷耗材都做 6 校准要多久？ ▼
不需要每次都做全部 6 项，按场景分配：

每卷耗材（5-10 分钟）：
→ First Layer + Flow Rate

每换一种耗材类型（25 分钟）：
→ 上述 + Temperature Tower + Retraction Test

装机/换热端后（1.5 小时）：
→ 全部 6 项 + 等待降温重启时间
Q3 KLP1 沿用基线 PA = 0.05 还需要重做 PA 校准吗？ ▼
取决于你的质量要求：

普通件：基线 0.05 可以直接用，无需重做

工业精密件：建议做一次 PA 校准确认，因为：
→ 不同品牌 PLA 流动性有差异
→ BMG 齿轮磨损后实际表现不同
→ 热端积碳会影响压力曲线

PA 校准件只需打印 30 分钟，建议装机时做一次存档。
Q4 流量校准和 PA 校准先后顺序哪个重要？ ▼
必须先做流量校准，再做 PA 校准，原因：

① Flow Rate 决定实际出料量
② 如果流量不准（偏高/偏低），PA 值测出来也是不准的
③ PA 校准件在流量错误时无法正确判断

正确顺序：
First Layer → Temperature → Flow Rate → PA → Retraction → VER
Q5 SuperSlicer 和 OrcaSlicer 校准结果通用吗？ ▼
基本通用，但需注意：

通用参数：
→ Z offset、Flow Rate、Pressure Advance
→ 这些是打印机/耗材物理参数， slicer 通用

Slicer 专属参数：
→ Temperature Tower、Retraction Test 建议在目标 slicer 中做
→ 因为不同 slicer 的温度补偿和回抽算法略有差异

Andy 推荐：确认使用 SuperSlicer 后，所有校准在 SS 中完成。
Q6 校准件 STL 文件在哪下载？ ▼
SuperSlicer 内置校准件，无需额外下载：

1. 打开 SuperSlicer
2. 点击顶部菜单 校准 (Calibration)
3. 选择对应校准项
4. 点击 Slice 即可生成

如果需要独立 STL 文件，可从以下来源获取：
→ AndrewF001/Kalibration
→ Printables PA Calibration
Q7 校准件打印失败怎么办？ ▼
常见失败原因和解决方案：

首层不粘：
→ 降低 Z offset 0.02-0.05
→ 清洁热床（酒精/洗洁精）
→ 调高热床温度 5°C

拉丝严重无法判断：
→ 先用基线参数打印一版
→ 确认喷嘴温度正常后再做温度塔

中途堵料：
→ 检查耗材是否受潮
→ 降低打印速度 20%
→ 清理喉管和喷嘴
Q8 工业件客户件（医疗/航空）需要额外校准吗？ ▼
是的，需要更严格的校准体系：

基础 6 项（必须）：
→ 上述全部完成并记录数据

额外建议（行业认证）：
→ 张力环校准（确保力学一致性）
→ 尺寸精度测试（打印标准件量测）
→ 温度循环测试（模拟环境变化）
→ 耗材批次记录（可追溯性）

⚠️ 医疗/航空件建议联系客户确认具体标准（ISO 13485 / AS9100D 等）
Q9 KLP1 走 Voron Tap 后哪些校准要重做？ ▼
Voron Tap 是磁吸式喷嘴，切换后需重做：

必须重做：
→ First Layer（Tap 安装高度变化）
→ Z offset（BL Touch 触发点改变）
→ Pressure Advance（热端质量改变）

建议复查：
→ Temperature Tower（热端散热变化）
→ Retraction Test（回抽距离可能需要调整）

可能不变：
→ Volumetric Extrusion Rate（BMG 未变）
Q10 校准一次能管多久？ ▼
取决于使用场景：

参数稳定周期
Z offset每卷耗材
Flow Rate每卷耗材
Temperature每批耗材
Retraction每种耗材
Pressure Advance3-6 个月

Q11 不同耗材要分开做校准吗？ ▼
是的，不同耗材必须分开校准：

必须分开的：
→ PLA / PETG / ABS：温度差异大（ABS 需 240-260°C）
→ Retraction：PETG 回抽通常比 PLA 长 1-2mm
→ Flow Rate：不同品牌密度不同

可以共用的：
→ Pressure Advance：基线 0.05 可作为起点
→ Z offset：通常差异不大

SuperSlicer 支持多组 filament profile，记得为每种耗材保存独立配置。
Q12 校准数据怎么记录保存？ ▼
Andy 推荐记录方式：

1. SuperSlicer 配置文件
→ 保存为独立 filament profile（如 PLA-Inland-20240115）

2. 纸质/电子记录
→ 记录日期、耗材批次、测量值、校准结论

3. GitHub 仓库
→ 导出 printer.cfg 和 filament profiles 到仓库
→ 仓库地址
