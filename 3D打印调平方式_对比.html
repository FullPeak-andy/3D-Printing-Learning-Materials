🔘 机械探针 · BLTouch / CR-Touch

原理：探针内部有弹簧+磁铁+霍尔传感器。下垂时探针从吸起位置下落，碰到床后被推开触发霍尔，向上吸起。Klipper 看到探针触发 → 记录当前 Z 高度 = "床到这个点的高度"。

🎬 动画演示：探针下落 → 触发 → 收回

📌 关键参数

  • 测量精度：±0.005 mm（触发式测量）
  • 单点测量时间：~1.5 秒（下降+触发+上升）
  • 常见测点数：16-49 点（5×5 网格 25 点）
  • 价格：¥30-80（BLTouch） / ¥50-100（CR-Touch）
  • 寿命：约 1-3 年（机械部件多，易磨损）

✅❌ 优缺点

优点

  • 精度高，触发测量准
  • 不挑喷嘴/床材质
  • 价格便宜，30 元起
  • 改装难度低（替换原 Z 限位）
  • 开源生态成熟（Klipper 原生支持）

    • 缺点

  • 有机械动作 → 易磨损/卡死
  • 速度慢（每个点 1.5 秒）
  • 展开/收回易撞打印件
  • 探针偏移 / 撞坏要重新校准
  • 非接触场景下不工作

    • 🎯 适用机型

    KLP1 (你的) Voron 2.4 Ender 3 Prusa MK4 Bambu A1 mini Bambu P1S
    💡 Andy 你的 KLP1 v3 修复版：用 MKS_THR.cfg 里的 [probe] 段配置（pin: !MKS_THR:gpio21, x_offset: 12.85），实际生效 z_offset 在 SAVE_CONFIG 段（当前 1.780）。

    🟠 电容接近开关

    原理：金属圆柱体作为感应面，靠近金属/电容物体时电容值变化 → 内部电路识别后输出开关信号。装在挤出机上（不在打印头上方）→ Z 接近时触发。Kobra 2 / Ender S1 / K1 SE 等用这个方案。

    🎬 动画演示：探针下降 → 电容变化 → 触发

    📌 关键参数

    • 测量精度：±0.02 mm（模拟信号）
    • 单点测量时间：~0.5 秒（无机械动作）
    • 感应距离：1-4 mm（可调）
    • 价格：¥15-30（FYSETC 套件 ¥15.3）
    • 触发条件：必须用金属喷嘴，不能全塑料

    ✅❌ 优缺点

    优点

  • 无机械动作 → 不易坏
  • 速度快 3 倍（vs BLTouch）
  • 价格便宜（15 元起）
  • 免维护（不展开/收回）
  • 可以做到非接触调平

    • 缺点

  • 必须金属喷嘴（塑料喷嘴失效）
  • 受温度影响（喷嘴冷热变化 → 探针偏移）
  • 不同热床材料（PEI/玻璃/磁铁）探测距离不同
  • 装在挤出机上 → 离热源近 → 易受热形变
  • 精度比 BLTouch 低 4 倍

    • 🎯 适用机型

    Anycubic Kobra 2 Ender 3 S1 Creality K1 SE Voron 改装 FYSETC 通用套件
    💡 FYSETC 这个套件：淘宝 ¥15.3，针对 Kobra 2 设计的挤出机自动调平。KLP1 也可以改装（替换 BLTouch），但注意 KLP1 已经用 BLTouch 测得挺好，没必要为这个省钱。

    📐 应变片 (Strain Gauge)

    原理：应变片贴在热床和 Z 轴支架之间。热床微微受 Z 压力 → 应变片电阻变化 → 主控检测到变化 → 记录 Z 高度。**测的是物理形变**，所以不挑喷嘴/床材质。拓竹 P2S 用了 36 点应变片（覆盖整个床面）。

    🎬 动画演示：喷嘴接近 → 热床形变 → 应变片触发

    📌 关键参数

    • 测量精度：±0.001 mm（物理测量）
    • 单点测量时间：~0.3 秒（无展开）
    • 常见测点数：4 点 / 9 点 / 36 点
    • 价格：内置（机器自带）
    • 寿命：与机器同寿（无机械动作）

    ✅❌ 优缺点

    优点

  • 测的是物理形变，最直接
  • 不挑喷嘴/床材质（金属/塑料都行）
  • 速度极快（0.3 秒/点）
  • 无展开/收回动作
  • 36 点全覆盖（拓竹 P2S 旗舰方案）
  • 不占喷嘴附近空间

    • 缺点

  • 内置方案，不便后期改装
  • 需要重新设计 Z 支架（拓竹/创想新机用）
  • 受温度影响（金属热胀冷缩）
  • 点少时（4 点）精度反而不如 BLTouch
  • 校准算法复杂（需要学习材料弹性模量）

    • 🎯 适用机型

    Bambu X1E Bambu P2S (36点) Creality K1 Max Creality K1 Snapmaker 2
    💡 拓竹 P2S 36 点应变片 = 2025 年消费机最高调平方案。K1 Max 你那台也是应变片（4 点），所以你 K1 Max 的"Z 校准"按钮其实就是应变片调平。

    🌀 涡流探针 · BTT Eddy / Beacon

    原理：探头发出高频交变磁场 → 金属（喷嘴/热床）产生涡流 → 反过来影响探头线圈 → 检测线圈参数变化 → 算出**精确距离**。和 BLTouch 不同，**完全不接触**，而且可以连续测距（不只是 0/1 触发）。Voron 改装圈的"调平之王"。

    🎬 动画演示：涡流感应 → 距离变化 → 连续测距

    📌 关键参数

    • 测量精度：±0.001 mm（模拟测距）
    • 单点测量时间：~0.1 秒（连续扫描）
    • 感应距离：0-4 mm（连续可调）
    • 价格：¥150-400（BTT Eddy ¥150 / Beacon ¥350）
    • 寿命：与机器同寿（无机械）

    ✅❌ 优缺点

    优点

  • 不接触 → 永远不会撞坏
  • 测距不是 0/1 → 可连续扫描
  • Z 不用下来 → 速度极快
  • 改装潜力大（Voron 改装主流）
  • Klipper 原生支持（scan_mesh）
  • 可以测热床平整度 + 落差

    • 缺点

  • 价格贵（150+）
  • 需要金属喷嘴（不能全塑料）
  • 受温度影响（线圈电阻变）
  • 需要预热稳定（冷热差 0.05mm）
  • 改装难度中等（要重新接线）

    • 🎯 适用机型

    Voron 2.4 标配 Voron Trident K1 Max 改装 Ender 3 改装 KLP1 改装可行
    💡 拓竹 Beacon 同原理：Beacon 是 Voron 团队自己设计的涡流探针（¥350）。BTT Eddy 是大厂克隆版（¥150），Klipper 兼容。**Voron 2.4R2 必装 Beacon**。

    🎯 Andy KLP1 升级路径建议

    你 KLP1 v3 修复版现在用 BLTouch，已经能跑出不错的效果。如果要"再上一档"，按成本/收益排序：

    ① 现状先用好（不动硬件） ¥0

    把 BLTouch 调到极致：z_offset PROBE_CALIBRATE 准 + 输入整形配 ADXL345 + 6×6 mesh + KAMP 算法。质量能到拓竹 A1 水平。

    ② 改装 BTT Eddy 涡流探针 ¥150

    **最推荐改装路线**。替换 BLTouch，Klipper 原生支持，精度提升 5 倍，速度提升 15 倍。Voron 改装主流方案。

    ⚠ 注意：Eddy 需要金属喷嘴（KLP1 标配就是金属），改装需要重新接线。

    ③ 改装 Voron Beacon 涡流 ¥350

    Beacon 比 Eddy 贵 2 倍但**算法更优**（Voron 团队自研，scan_mesh 算法比 Klipper 原生更精细）。Voron 2.4R2 标配。

    ⚠ KLP1 装 Beacon 需要重新设计打印头支架（3D 打印一个），改装工程量比 Eddy 大。

    ❌ 不推荐：改装电容接近开关 ¥15

    便宜但**升级意义不大**：精度比 BLTouch 差 4 倍，受温度影响大，需要金属喷嘴。你已经有 BLTouch 了，没必要换这个。

    ⚠ 除非是 Kobra 2 那种**原装没调平**的机器，否则用不到。

    ❌ 不推荐：改装应变片 ¥500+

    应变片要重新设计 Z 支架（拓竹/创想新机自带），KLP1 改造工程量巨大。**机器架构不匹配**，不现实。

    🚀 最佳策略

    短期：把 BLTouch + PA + z_offset 调到极限
    中期：改装 BTT Eddy 涡流（Klipper 原生支持最稳）
    长期：上 Voron 2.4R2 + Beacon 涡流（终极方案）

    🌳 调平方式决策树

    不知道选哪种？按场景回答问题：

    Q1: 机器是新机还是改装？

    • 新机 → 看 Q2
    • 改装 → 看 Q3

    Q2: 新机是消费级还是 DIY 级？

    • 消费级（拓竹 A1 / P1S / 创想 K1）→ 机器自带什么用什么
    • DIY 级（自己攒机）→ BLTouch（最稳）或 Eddy（最先进）

    Q3: 改装预算？

    • ≤¥50 → 电容开关（但 KLP1 没必要）
    • ¥50-200 → BLTouch（成熟稳定）
    • ¥200+ → BTT Eddy（Voron 改装首选）
    • ¥300+ → Voron Beacon（终极方案）
