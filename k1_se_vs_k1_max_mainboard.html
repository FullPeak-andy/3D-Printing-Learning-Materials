✅ 核心结论：两台机器的上位机 SoC（Ingenic X2000E） + 三块 MCU（GD32F303RET6 + GD32F303CBT6 + GD32E230F8P6）完全相同。 也就是说主板 PCB 走线、芯片布局、硬件接口基本同源（同源 PCB 修订版 CR-K1-MB-V3R2）。
💡 性能参考：Ingenic X2000E 是 2020 年君正推出的多核异构跨界处理器，典型功耗 <380mW。1.2 GHz 双核 MIPS + 240 MHz 实时核的组合， 跑 Klipper + LVGL 触摸屏完全够用，但跑大模型 / 复杂图像识别是绝对不行的——这就是为什么 K1 Max 的 AI 摄像头和 AI LiDAR 都需要专用处理。

💡 采购启示：如果你的 K1 Max 主板坏了，不要去买 K1 SE 主板来换（刷错固件锁机），必须买 K1 Max 专属主板。 但如果只是某个 MCU 或模块坏了，这三块 MCU 芯片 + 断料器 + 步进驱动 + 触摸屏都是可以互换的。
🟢 同源优势（K1 SE 借鉴 K1 Max）
  • PCB 走线相同，从 K1 Max 主板照片就能 100% 复刻 K1 SE 主板维修
  • 三块 MCU 芯片相同，买 GD32F303RET6 / GD32F303CBT6 / GD32E230F8P6 都是同型号替换
  • 断料器模块通用，闲鱼 ¥15-30 任意买（K1/K1C/SE/Max 互通）
  • 5 颗 TMC2209 步进驱动通用，K1 SE 二手主板单买驱动也划算
  • 4.3 寸触摸屏通用，Max 屏坏了能买 SE 屏顶替
🟠 必须专属的部件
  • 主板整套（SE/Max 不能互换）
  • Nozzle 适配板：SE = LC-V13（短），Max = C-V13（长）
  • Z 探针：SE 光电 vs Max 机械微动
  • 机架：220³ vs 300³，尺寸完全不同
  • 电源：350W vs 1000W，加热能力差 2.86×
🔴 维修踩坑点
  • 刷固件前先看 EEPROM 签名，刷错同系列板子也会锁机
  • EEPROM ID 与外壳绑定，换主板后必须重新刷机
  • 电机参考电阻不同（0.08 vs 0.1），互换 TMC2209 没问题但电流配置需改
  • 风扇 PWM 曲线不同，刷错固件风扇要么不转要么狂转
  • Nozzle 适配板换错型号会触发热端温度异常

🎯 主板故障排查顺序
  • ① 屏幕不亮 → 查电源（350W vs 1000W）
  • ② 屏幕亮但 Klipper 不启动 → 查 eMMC / 刷固件
  • ③ 步进电机不动 → 查 TMC2209 + MCU0（GD32F303RET6）
  • ④ 热端不加热 → 查 Nozzle MCU（GD32F303CBT6）
  • ⑤ 热床不加热 → 查热床 MCU（GD32E230F8P6）
  • ⑥ TR0116 误报 → 查断料器（机械弹簧臂）
  • ⑦ Z 探针不触发 → 区分光电/机械
⚡ 创想反 DIY 设计
  • EEPROM 签名 + 机壳绑定，换主板必须重新配对
  • 新版 CreOS 固件隐藏了断料检测菜单（你的 K1 Max 现在就是）
  • 创想云不开放断料检测、Z 偏移、流量等底层设置
  • 必须刷 Root + Klipper才能解锁所有功能（V1.3.5.5+）
  • 想完全 DIY → 走社区版 K1-Klipper 或 cryoz/klipper
📌 给你 K1 Max 的实操建议：
  • 你目前 TR0116 误报 → 先查断料器机械结构（清洁弹簧臂润滑 + 万用表测微动开关通断）
  • 想关闭断料检测 → 创想云 + 屏幕菜单都关不了，物理断开排线是最稳方案
  • 想玩 Klipper 完整功能 → 刷 V1.3.5.5+ 固件 + 走社区版 K1-Klipper
  • 主板备件 → 闲鱼 K1 Max 主板 ¥150-250，三块 MCU 单独买芯片更划算
