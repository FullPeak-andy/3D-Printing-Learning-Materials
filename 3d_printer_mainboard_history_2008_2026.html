8-bit AVR 时代（2008-2014）

Arduino Mega 2560 主导 + Marlin 固件奠基 · RepRap 哲学大众化
📋 阶段概述
CPU：AVR ATmega2560（8-bit，16 MHz，256KB Flash + 8KB SRAM）
代表架构：Arduino Mega 2560 + RAMPS 1.4 shield + Pololu A4988 模块化驱动
固件：Marlin 1.0（2011-08-12 GitHub 发布）→ Marlin 1.1.x
核心创新：模块化（Shield 堆叠）+ Pololu 步进驱动（焊/拆皆可）+ 12V 电源体系
商业引爆点：Prusa i3（2012）成为全球 DIY 3D 打印"参考设计"
⚠️ 局限：8KB RAM 装不下高级功能（断料续打/线性推进需外挂）；16MHz CPU 在 80steps/mm × 50mm/s 时已接近极限；不支持 128/256 细分（TMC 必需）

🔑 5 项关键历史改进
① RepRap 开源哲学建立（2005-2007）
Adrian Bowyer 在英国巴斯大学发起 RepRap 项目（replicating rapid prototyper）→ 第一台 Darwin（2007）能"自复制" 50% 的塑料部件 → 树立"开源 + 可自复制 + 持续迭代"的社区文化
② Marlin 固件发布（2011-08-12）
从 Sprinter/grbl 衍生，Scott Lahteine 主导，专为 RepRap 设计。GPLv3 协议；支持 G-code 150+ 命令；PID 控温；Marlin 后成为 8 位 AVR 时代的"事实上标准"，到 2026 年仍是出货量最大的固件
③ RAMPS 1.4 Shield 标准化（2011-08-04）
RepRap Arduino Mega Pololu Shield，Arduino Mega 上插一块即得完整 3D 打印机控制。Pololu A4988 步进驱动模块化（焊/插随意）→ 大幅降低 DIY 门槛
④ Pololu A4988 / DRV8825 步进驱动
Allegro A4988（16 细分 / 2A）和 TI DRV8825（32 细分 / 2.5A）成为标配，A4988 的插拔模块化（5 个槽）让 8 位 AVR 板可以支持 5 轴，但发热大、噪音大，机械声像"打嗝"
⑤ Prusa i3 商业化引爆（2012-2014）
Prusa Research 将 RepRap 设计标准化为 i3（cartesian + 床式 Y），Joseph Prusa 开源所有固件和 BOM → 出现 Anet、Creality、Anycubic、Tronxy 等"Prusa 克隆"厂商，3D 打印进入消费市场

🔑 5 项关键历史改进
① Klipper 革命：MCU + SBC 双芯架构（2016）
Kevin O'Connor 发布 Klipper 固件 → 上位机（树莓派/任何 Linux SBC）做路径规划 + 下位机（MCU）做实时硬件响应。这是 3D 打印架构的根本性变革，让 175K steps/s 的高细分驱动成为可能（8-bit AVR 跑 50 steps/mm × 50mm/s 已极限）
② 32 位 ARM Cortex-M3 100MHz 首次普及
LPC1768 替代 AVR：512KB Flash（2 倍 AVR） + 64KB SRAM（8 倍 AVR） → 装得下"断料检测 / 断电续打 / 喷嘴泊车 / 高级暂停 / S 曲线加速 / Linear Advance"等高级功能
③ TMC 驱动首次出现（TMC2100）
Trinamic 推出 TMC2100（首代 StealthChop 静音）→ 256 细分 + 静音运行，结束了 A4988/DRV8825 的"打嗝时代"。但当时价格高（一片 ¥80+），未普及
④ Marlin 编译环境迁移：Arduino IDE → PlatformIO
Marlin 2.0 起不再支持 Arduino IDE（1.1.x 仍支持），需要 PlatformIO 或 VSCode + PlatformIO 插件。门槛提高，但 32-bit 多文件管理能力大幅提升
⑤ Creality 崛起（2015-2017）
深圳拓想（Creality）成立于 2014 年，2015 年推出 CR-10，2016 年 Ender 1/2 试水，2018 年 Ender 3 一炮而红（¥999 全套）→ 3D 打印进入"消费级"时代

🔑 5 项关键历史改进
① STM32F407 Cortex-M4 168MHz 主流化
意法半导体 STM32F4 系列：168MHz 单精度浮点 / DSP 指令 / 512KB Flash + 192KB SRAM → 性能是 LPC1768 的 1.7 倍，价格相当（¥30-50/片）→ 几乎一夜替代 LPC1768 成为开源 3D 打印主板首选
② TMC2208/2209 板载 + UART 模式
TMC2208（2017）/ TMC2209（2019）板载（替代 Pololu 模块）→ 256 细分 + StealthChop 静音 + UART 通信。StealthChop 让 3D 打印噪音从 50dB 降到 30dB，Voron 静音的硬件基础
③ Klipper 上位机成熟（树莓派 + Klipper）
Klipper 0.x → Klipper 2020 期间，MCU 通过 USB 连树莓派，配置由树莓派上的 printer.cfg 决定 → 无需重编译，浏览器（OctoPrint/Mainsail/Fluidd）直接编辑 → 软件迭代速度比 Marlin 快 10 倍
④ Voron Design 团队成立 + V2.4 发布（2020）
Voron-2 GitHub 4.4k ⭐（2026 仍是开源 3D 打印 Top 1）→ 4 独立 Z 电机 + 万向节（gantry joints） + 闭环皮带 → "无妥协"理念重新定义 3D 打印质量天花板。Voron 2.4 必装 Klipper，反向推动 Klipper 成为主流
⑤ BTT（拓竹）2018 深圳成立
深圳必趣科技（BTT）2018 成立，专注 3D 打印主板 → SKR V1.3（2019）→ SKR V1.4（2020）→ 一举吃下 Voron 主板市场。同期 MKS（创客基地）、Fysetc（飞思策）三足鼎立

🔑 5 项关键历史改进
① 8 步进槽成为 Voron 标配
Voron 2.4 4Z（4 个 Z 电机）+ X/Y + 挤出机 + 备选第 6/7 槽 → 主板需要 8 步进槽。BTT Octopus / Fysetc Spider 引领，4 步进槽板（如 SKR V1.4）被淘汰出 Voron 主板选项
② Input Shaper 加速度革命（Klipper 2021）
Klipper 引入 Input Shaper + ADXL345 加速度计 → 测量 X/Y 共振频率 → 主动消除打印头共振 → 打印速度从 80mm/s 提升到 200-500mm/s，质量不降。这是 Klipper 第一次在"功能层"超越 Marlin
③ CAN-FD 工具头板革命
BTT EBB / Fysetc EBB SB-CAN V1.0（2021）→ 工具头板通过 4 线（CAN-H/CAN-L/GND/+24V）连接主板，工具头端的 8+ 根线（加热棒/热敏/风扇/挤出/探针/灯）全部本地处理 → "去皮线"成 Voron 标配
④ TMC2209 全面替代 TMC2208
TMC2209 比 TMC2208 多 0.5A 电流（2.0A vs 1.5A RMS） + StallGuard 失步检测 → Klipper 可识别打印头碰撞 + 自动 PA 调整。2208 在大流量挤出时发热严重，逐步淘汰
⑤ Marlin 2.0.x 稳定 + 32-bit 全面接管
2021 年 Marlin 2.0.9 稳定版发布，2.1.x 开发版 2022 推出，32-bit ARM Cortex-M4 成为 Marlin 主力支持。AVR 8-bit 仍维护（限于低功能），商业市场全面转向 32-bit
