1️⃣ KMS-V1.0 是什么？

你在 B 站看到的视频展示的 GitHub 仓库：muzixiaoyang/KMS，作者是 muzixiaoyang。

📌 一句话定位：KMS = Klipper Multi-Color System，基于 Happy Hare TYPE-B 软件方案 + 独立通道硬件设计（不依赖选择器/舵机），KLP 圈中国开发者 muzixiaoyang 的开源项目。

🔧 核心特性

  • 独立通道设计：每个料管独立路径，没有传统 MMU 的选择器/servo，简化控制流程，提高可靠性
  • Happy Hare TYPE-B 软件：使用 moggieuk 开发的通用 MMU 驱动，社区最主流的 Klipper 多色固件
  • 温度湿度监控：PCB 集成传感器，监控密封盒内环境（防潮）
  • KMS-Buffer 同步系统：保证打印机端和 KMS 端料丝传输同步
  • 电动卷绕器：模块化设计，方便料管回收
  • MMB-Board（Beta）：配套 BigTreeTech 控制板
  • 3D 打印外壳：含亚克力透明窗，可视化外观
  • 开源协议：GPL-3.0

💰 商业化情况

Trianglelab（深圳三绿）有卖成品套件，标价 $112 / 折后 $106.40，不含打印件（需自打 STL）。注意：商业套件是别人基于开源方案量产的，不含税，国际运费贵。

4️⃣ 给 Andy 哥的建议

❌ KMS 现阶段不适合你

KMS 是给 Klipper + CoreXY（Voron/改造机）设计的。KLP1 是 i3 + Marlin，硬件方案不同，装不上也跑不动。

⚠️ ERCF 也不适合 KLP1

ERCF 整套（Voron + Klipper）你要是上 KLP1 等于把宝马发动机塞到 QQ 里。但是 Voron 圈是开源 MMU 老大，值得看视频学设计思路。

✅ K1 改造机能上

CoreXY + 改装 Klipper + 加 StealthChanger 码头可以玩 ERCF / KMS / 3MS。预算 +200-500 元（材料 + 控制板）就能体验。改造成本远低于买大鱼 SPA 4 色机。

❌ 大鱼 SPA 4 色机不用 ERCF

大鱼 SPA 是 i3 + 4 色 Z 轴切换（料塔式），跟 ERCF 走的是完全不同的硬件路线。装了 ERCF 也用不上。

✅ 现在该做的

看 B 站「ERCF 实装」+「Happy Hare 教程」+「KMS 视频」3 个入门视频，先搞懂原理。等 K1 改造机真上了 Klipper，再决定上哪个方案。
