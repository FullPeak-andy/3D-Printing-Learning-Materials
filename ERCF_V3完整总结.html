📖 一句话定位

ERCF V3 是 Voron 圈最成熟的开源多色换料器（MMU），给单直驱工具头（Stealthburner）提供 8 通道（可扩 12 色）自动换料。ERCF v1.1 → v2 → v3三代演进，从 Ette 一个人维护变成 Carrot Collective 社区团队主导，是当前 Voron + Klipper 多色方案的事实标准。

GitHub：https://github.com/Carrot-collective/ERCF_v3

协议：GPL-3.0（完全开源可商用）

团队：Carrot Collective（兔子多色团）· 原作者 Ette 背书

🏗️ 项目架构（4 大子模块）

🥕
ERCF（Enraged Rabbit Carrot Feeder）主换料器
8 通道（标准）/ 12 通道（扩展）多色换料主机。多进单出，单直驱工具头配合。集成选择器 + 编码器 + 弹簧伺服。是整个系统的核心。
🐇
ERCT（Enraged Rabbit Cotton Tail）集成缓冲器
集成式被动料丝缓冲器，直接装在 ERCF V2/V3 出口。空间优化 + 路径阻力降低 + 内置预门传感器 + 每路独立 NeoPixel 状态灯（"bling"）。
✂️
ERF（Enraged Rabbit Filametrix）工具头切刀
装在 Stealthburner 工具头上的切刀，退料时直接切掉问题料尖。免去 tip-forming（料尖成型）的调参噩梦。每次切出干净料尖，零堵塞。
🧹
ERPS（Pellet Purge）颗粒料清除系统
用颗粒料（pellet）清除系统，免去擦料塔（wipe tower）。目前仅支持 Voron V2 框架。持续开发中。

🧠 固件生态（Happy Hare 驱动）

ERCF V3 必须配合 Happy Hare 固件使用，这是 moggieuk 开发的 Klipper MMU 通用驱动。

Happy Hare 核心功能

  • 工具头传感器支持 + 无传感器加载/卸载
  • 同步运动 — 挤出机 + 齿轮电机一起移动
  • EndlessSpool — 自动识别空门并重新映射工具（不会中断打印）
  • 改进的日志 — 控制台 + 独立 mmu.log 文件
  • 可视化 — 加载/卸载序列动画
  • 运行时配置 — 新命令在运行时改配置
  • Spoolman 集成 — 耗材管理
  • Klipperscreen / Mainsail / Fluidd UI
  • 支持几乎所有 MMU：ERCF / Tradrack / Box Turtle / Angry Beaver / Night Owl / 3MS / KMS / Open AMS / PicoMMU / MMX / Custom

🎯 给 Andy 哥的判断

❌ KLP1 完全装不上

KLP1 是 i3 + Marlin 架构，ERCF 是给 CoreXY + Klipper 设计的。机械结构（CoreXY 运动学）、固件（Klipper）、通道数（8 路）都跟 i3 不兼容。

❌ 大鱼 SPA 也不兼容

大鱼 SPA 4 色机是 i3 + Z 轴料塔式（4 个挤出头上下切换），跟 ERCF 单直驱+换料器路线完全相反。

✅ K1 改造机能上

CoreXY + 改装 Klipper + 加 Stealthburner 工具头 + ERCF 是 Voron 圈标准玩法。改造成本 +200-500 元（材料 + 控制板 + 切刀），远低于买大鱼 SPA 4 色机。

⚠️ 现阶段不必买

K1 改造机还没定型（Voron 风格 CoreXY 改装方案还在评估），先看 B 站 3-5 个 ERCF 实装视频 + Happy Hare 文档，搞懂原理。

✅ 收藏到 GitHub

这个仓库是 Voron 多色圈的"圣经"，等 K1 改造机成熟后必看。现在先 Star 一下，留着参考。

💡 学设计思路

ERCF 的"单直驱+多通道换料"思路 vs 大鱼 SPA 的"多挤出头+Z 轴切换"思路，都是多色方案经典设计。学思路比买硬件更重要。
