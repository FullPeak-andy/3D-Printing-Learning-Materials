1️⃣ 拓竹 A1 "LiDAR" 真实拆解

拓竹 A1 宣传的"LiDAR"功能，实际上是3 个独立功能的营销概念：

📡

首层检测 = ToF 激光测距

芯片：VL53L1X（ST 意法半导体）
原理：发射红外激光，测量反射时间计算距离
精度：±1mm（4m 范围内）

📊

打印中检测 = ToF 测每层

原理：打印完一层后，用 ToF 测该层高度
对比预期高度，判断是否有打印问题
类似 Klipper 的 RESERVOIR 检测

🤖

AI 视觉 = 720P 摄像头 + 端侧 AI

硬件：720P USB 摄像头（约 $5）
AI：端侧推理（不依赖云端）
功能：飞溅检测、打印异常识别

💡 关键洞察："LiDAR"这个词很唬人，实际上是 ToF 传感器 + 摄像头 + AI 软件的组合。KLP1 完全可以用更低成本实现类似功能。

    3️⃣ 🅰️ 路线 30 分钟安装指南

    🅰️ 入门路线安装步骤

    1

    准备硬件

    • 购买 KLP1 兼容 USB Webcam（约 $10-20）
    • 推荐罗技 C270/C920 或任意 UVC 摄像头
    • 摄像头安装在热床旁边（俯视角度）

    2

    注册 Obico 账号

    • 访问 obico.io 注册免费账号
    • 免费账号：基础远程查看
    • 订阅 $4/月：AI 飞溅检测 + 高级功能

    3

    Moonraker 配置

    Moonraker 已集成在 KLP1 的 Klipper 中，无需额外安装。只需确保 Webcam 被 Moonraker 识别：

    # Moonraker 默认配置（printer.cfg 中已有）
[webcam server]
enable = true
stream_url: /webcam/?action=stream
snapshot_url: /webcam/?action=snapshot
flip_h = False
flip_v = False
rotation = 90  # 根据摄像头方向调整

# 测试 Webcam
# 浏览器访问：http://KLP1_IP/webcam/?action=stream
# 应能看到实时画面
    4

    Obico Klipper 集成

    # moonraker.conf 中添加 Obico 插件
[update_manager obico]
type: git_repo
path: ~/klipper_config/obico
origin: https://github.com/TheSpaghettiDetective/obico-server.git

# 或使用 Moonraker 的 plugin_manager 安装
# SSH 到 KLP1：
# ~/moonraker-env/bin/pip install obico

# 配置 Obico
# 在 Obico 网站添加打印机 → 获取 Server Link Code
# 粘贴到 KLP1 的 Moonraker → Plugins → Obico
    5

    Obico AI 飞溅检测

    • 订阅 $4/月后开启 AI 检测
    • 设置检测灵敏度（建议中等）
    • 设置异常通知方式（邮件/微信/Telegram）
    • 首次打印建议全程观察，训练 AI 模型
