📋 常见问题 FAQ

1. 为什么不勾 Supports binary G-code？ +
KLP1 使用 MKS 主板，其 Klipper 固件默认不支持二进制 G-code（M997 指令）。勾选此选项会导致切片输出的 G-code 在打印时解析错误。保持不勾，使用标准 ASCII G-code 确保 100% 兼容。
2. Arc fitting 启用后会增加切片时间吗？ +
会有轻微增加（约 3-5%），但对于大多数模型影响可忽略。Arc fitting 将连续的短直线拟合为圆弧，可以减少 G-code 文件大小、提升打印流畅度。对于包含大量圆形特征的模型（如齿轮、管道），建议启用并设为 80-90% 精度。
3. Klipper 模式下 Start G-code 留空真的不会出问题？ +
不会出问题，前提是你的 Klipper printer.cfg 中正确配置了 START_PRINT 宏。该宏应包含：热床/喷嘴加热 → 巡边/调平 → 冲刷 purge line → 打印。SuperSlicer 会通过 {自有宏}START_PRINT... 参数传递温度信息。保持 SuperSlicer 留空可以让所有打印初始流程统一管理，修改 Klipper 配置后无需重新切片。
4. Travel lift 0.2mm 会让悬空下垂吗？ +
不会。Travel lift 是在移动前先抬升喷嘴 0.2mm，避免喷嘴刮蹭已打印的模型表面。这与打印过程中的悬空桥接（bridging）无关。KLP1 的 0.4mm 喷嘴配合 0.2mm 层高，桥接能力约 5-8mm，足够大多数工业件需求。启用 "Use ramping lift"（斜坡式抬升）比垂直抬升更能减少 Z 轴振纹（z-banding）。
5. 4010 风扇最低 30% PWM 是硬限制吗？ +
是的，这是 Sunon MF40100V2 的物理特性。低于 30% PWM（约 0.8V）时，风扇可能无法启动或运行不稳定（抖动、异响）。SuperSlicer 中的 "Minimum fan speed" 设置为 30% 确保风扇始终在稳定转速范围运行。如果需要更低转速的静音效果，考虑更换支持 5V 起始的风扇（如 Noctua NF-A4x10）。
6. 喷嘴 0.4mm 最大层高 0.3mm 是怎么算的？ +
层高与喷嘴直径的比例关系：
• 最小层高 = 喷嘴直径 × 5% = 0.4 × 0.05 = 0.02mm
• 最大层高 = 喷嘴直径 × 75% = 0.4 × 0.75 = 0.30mm

超过 75% 层高会导致挤出不均匀、层间粘合不良。0.4mm 喷嘴推荐常用层高：
• 精细：0.12-0.16mm（外观要求高的模型）
• 标准：0.20mm（平衡质量与速度）
• 快速：0.28mm（原型验证）
7. 加速度 4000 比 9000 慢很多，会不会打印时间增加？ +
不会显著增加。加速度主要影响起步/停止的响应速度，而不是匀速运动时的速度。KLP1 的 4000 mm/s² 加速度是经过硬件测试的安全值，可以：
• 减少电机抖动和共振噪音
• 降低机械磨损
• 提高打印稳定性（减少模型细节处过冲）

实际打印测试表明，4000 vs 9000 加速度的成品总时间差异通常 <5%，但打印质量差异明显。
8. Use ramping lift 真的会减少 z-banding 吗？ +
是的。Z-banding（Z 轴振纹）是由 Z 轴电机启停时的冲击力导致的垂直条纹。垂直抬升（垂直向上）时，冲击在每层重复发生，形成规律振纹。

Ramping lift（斜坡式抬升）将垂直抬升分解为多个小步，冲击力分散到多个微移动中，大幅减少振纹。配合 KLP1 的静音步进驱动，效果更明显。

10. 我用 Beta 2 切 Advanced 后"结束"按钮还是灰色？ +
这是 Beta 2 的已知 bug。按顺序尝试：
1. 勾选"选择此页面中的所有标准打印机"
2. 在打印机列表中点击任意一台机器
3. 按 F5 刷新 UI
4. 如仍不行，直接关闭 Wizard，后续在 Printer Settings 中手动完成配置

这个 bug 不会影响实际打印，只是 UI 交互问题。
11. Z 偏移为什么不在 SuperSlicer 设？ +
KLP1 使用 Klipper + BLTouch，z_offset 应在 Klipper printer.cfg 中管理：

[probe]
z_offset: 1.050

原因：
• Klipper 的探针校准更精确（多次采样取平均）
• 可以随时重新校准而不影响已有切片文件
• 与 Klipper 的 mesh leveling 系统无缝集成

SuperSlicer 的 Z offset 设为 0，让 Klipper 全权负责。
12. 工业件和日常件配置要分开保存吗？ +
强烈建议分开。两种场景的配置差异：

工业件：
• 精度优先：0.12-0.16mm 层高
• 低速稳定：加速度 3000，进给率保守
• 开启所有 G-code 钩子（进度上报、层监控）

日常件：
• 速度优先：0.2-0.28mm 层高
• 可用较高加速度
• G-code 钩子可简化

SuperSlicer 支持多配置文件导出，建议：
• KLP1_Industrial.ini
• KLP1_Prototyping.ini
