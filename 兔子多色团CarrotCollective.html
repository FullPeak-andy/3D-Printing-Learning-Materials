📖 兔子多色团是什么？

Carrot Collective（兔子多色团）是 Voron 多色打印生态里最硬核的团队之一，由多位核心贡献者组成，专注多色换料（Filament Changer）整套方案。

主仓库 ERCF V3（Enraged Rabbit Carrot Feeder）是当前 Voron 圈最成熟的开源 8 色 ERCF 方案，1,200+ stars。围绕 ERCF 还发展出切刀、排料、缓冲等 4 个配套项目，加上炸脖龙 Jabberwocky 这个 Affiliate 高端工具头，构成完整的多色打印工具链。

💡 给 Andy 哥的使用建议

  • 先收藏 ERCF V3 仓库：主仓是整套方案的总入口，BOM 清单 / 打印件 STL / 装配指南都在这里
  • 想装 KLP1？装不上：KLP1 是 i3 架构，ERCF 是给 CoreXY（Voron/改造机）设计的
  • K1 改造机有可能：CoreXY 架构 + Voron StealthChanger 码头可上，但需要上 ERCF + 切刀 + 工具头全套
  • 大鱼 SPA 4 色机：方案不同：大鱼是 i3 + 4 色挤出，硬件方案是 Z 轴切换+挤出塔，跟 ERCF 走的是完全不同的路线
  • GitHub 看完后去 B 站：搜"兔子换料" "ERCF" 有大量中文实装视频，比单看文档上手快
