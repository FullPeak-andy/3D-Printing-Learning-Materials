🔍 远程验机 6 步（自提 / 视频验机）

1️⃣ 卖家拍整体视频（30秒）

要求：通电 → 开机 → 屏幕显示正常 → 手动归零 XYZ → 加热床升温到 60°C → 风扇转动。要求卖家发完整视频，确认无报错。

2️⃣ 拍 4 色送料测试

要求：4 个 AMS 槽分别送料到喷嘴 → 切换 3 次 → 检查是否漏料 / 堵料 / 拉丝。这是木子 AMS-LITE 的核心功能，必须验证。

3️⃣ 拍切刀擦嘴测试

要求：切刀切料 → 擦嘴动作流畅 → 切面平整。如果切面毛糙或擦嘴卡顿，说明调试不到位。

4️⃣ 试打标准模型（方块 + 小人）

要求卖家打 20×20×20mm 实心方块 + 拓竹测试小人（小物件就行），看表面质量、层纹、悬垂。如果卖家不愿意打，说明机器可能有暗病。

5️⃣ 拍主板 + 驱动特写

要求：拍 STM32F103 主板照片 → 看电容有没有鼓包 → 看 TMC2209 散热片是否齐全 → 看接线是否整齐 → 闻有没有焦味。

6️⃣ 询问 3 个关键问题

① 用了多久？为什么出？（正常回答是"升级 / 换机 / 玩腻了"）
② 是否原包装？发票？保修凭证？（虽然 ¥366 不指望保修，但要问清楚）
③ 能不能走闲鱼担保交易？（不接自提就要求担保，避免扯皮）
