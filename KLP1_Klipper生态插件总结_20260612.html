Klipper 生态数据流架构（KLP1 视角）
架构关键点
• Klipper 拆 host + MCU：算法在 host（树莓派/Linux），MCU 只管"按时钟发脉冲"
• Moonraker 是中枢：所有 Web UI / 插件 / 外部服务都通过它和 Klipper 通信
• 插件分两类：Moonraker 插件（用 API 通信）vs Klipper 模块（在 host Python 里运行）
• UI 三选一：Mainsail（深色+功能多）/ Fluidd（轻量）/ KlipperScreen（触屏）
• 改配置不重编：.cfg 文件改完重启服务即可，不用刷固件

T
Telegram Bot
❌ 未安装
把 Klipper 接到 Telegram。打印开始/完成/失败发消息到你的 TG，也能发指令（如 /status, /pause）。免费、不需公网 IP。
你可能不需要：KLP1 在家里网络，TG 通知用得到，但不如 Obico 强大。先用其他方式。
O
Obico for Klipper
❌ 未安装
云端 AI 监控。摄像头 + AI 检测"打歪了/塌了/翘边"等故障，自动暂停打印 + 发通知。比 TG 强大 —— 主动发现问题。
免费额度：每月 10 小时，超出付费（$4-8/月）。家用偶尔打够用。
关键功能：AI 故障检测 / 远程看摄像头 / 远程发送 GCode。
E
OctoEverywhere
❌ 未安装
免费远程访问。免公网 IP、免端口映射，从任何地方浏览器打开 klp1.octoeverywhere.com 就能控制打印机。AI 检测免费。
对你价值：比 Obico 远程访问更稳定，AI 检测免费。最推荐先装这个，你晚上工作 / 上班都能看 KLP1。
M
Mobileraker
❌ 未安装
Klipper 官方手机 App（开源 / 跨平台）。iOS + Android 都有。简洁直接 —— 看温度、查打印进度、暂停/恢复、发送 GCode。必须装 Moonraker 0.8+，你 KLP1 满足。
优点：开源、免费、无广告、轻量。
缺点：功能相对少，UI 朴素。
A
OctoApp
❌ 未安装
第三方手机 App（iOS 优化更好）。功能比 Mobileraker 多：内置 GCode 查看器、电源控制插件支持、OctoEverywhere/Obico 集成。免费试用 + 内购解锁（USD 12.99）。
优点：功能多、UI 漂亮、iOS 用户体验最好。
缺点：不是开源、需付费解锁全部功能。
怎么选？
• 想要开源免费：装 Mobileraker（跨平台）
• 想要iOS 体验最好且不差钱：装 OctoApp
• 想要远程访问不挑设备：装 OctoEverywhere（浏览器访问，无需装 App）

📊 KLP1 总结
核心 5 件套 全部装好，4 个最新版（Mainsail / Fluidd / KlipperScreen / Moonraker），1 个可升级（Klipper 0.11 → 0.12）。
Crowsnest 摄像头最新版（前面用的就是它）。
6 个远程/通知/手机 App 插件 都没装 —— OctoEverywhere 性价比最高（免费 + AI 检测 + 远程访问）。

1️⃣ 先把首层问题搞定（PID 持久化 + z_offset 重测 + mesh 重测）

3️⃣ 最后考虑 Klipper 0.12 升级（需刷 MCU 固件，谨慎）
