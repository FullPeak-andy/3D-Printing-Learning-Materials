🔍 三件套识别（看图说话）

你拍的照片里，三个组件从左到右是：

🔩
加热块 + 喷嘴 + 喉管
Heater Block + Nozzle + Throat
位置：热端正中央主体
材质：铝合金（加热块）+ 黄铜（喉管）+ 铜/钢（喷嘴）
作用：
• 加热块：装加热电阻、热敏电阻的载体
• 喉管：引导料从上方进入喷嘴
• 喷嘴：0.4mm 出口，决定打印精度
温度承受：300°C 持续
价格：30-100 元（按材质）
🌡️
热敏电阻
NTC 100K Thermistor
位置：插在加热块侧面小孔
外观：玻璃珠（米黄色） + 2P 白插头
线序：红线（信号）+ 黑线（GND），无正负
作用：测温，告诉主控现在热端多少度
关键参数：
• 25°C 室温：~100KΩ
• 200°C：~0.3KΩ
• 250°C：~0.15KΩ
价格：5-15 元
⚠️ 你的这颗：已烧焦，必须换
⚡
加热电阻 / 加热棒
Heater Cartridge
位置：插在加热块中间大孔
外观：米黄色陶瓷方块 + 2P/3P 插头
线序：红/白线（无极性）
作用：电热丝通电发热，把热端升温到 200-260°C
关键参数：
• 电压：24V DC
• 功率：40-50W
• 阻值：~11-15Ω（24V/40W=14.4Ω）
• 升温：60-90s 到 200°C
价格：5-20 元
⚠️ 你的这颗：表面烧焦，建议测阻值
⚠️ 三个组件常见连带故障
堵头 → 料挤不出 → 加热持续 → 温度失控 → 热敏电阻烧毁 + 加热电阻外壳烧焦

建议：三个一起换（总共 20-30 元，比单独换省事）

🌡️ 热敏电阻：万用表测试 + 更换

📋 测试工具

需要的工具
• 万用表（电阻档 Ω 或 200K 档位）
• 一杯热水（80°C 左右，不烫手即可）
• 一杯常温水（25°C）

📝 测试步骤

1
万用表拨到电阻档
拨到 200KΩ 或 2MΩ 档。红黑表笔随便插哪根线都行，热敏电阻无极性。
2
测常温阻值（25°C）
玻璃珠放在室温 5 分钟，正常应该是 95-105KΩ。偏离这个范围就是坏的。
~100 KΩ ✅
3
测高温阻值（80°C）
把玻璃珠（只玻璃珠部分，别把线泡了）放进 80°C 热水 30 秒，正常应该降到 12-15KΩ。
~13 KΩ ✅
4
判断好坏
✅ 正常：常温 100KΩ，热水中下降 10 倍以上
❌ 开路：显示 OL 或 1. → 内部断了，必换
❌ 短路：显示 0Ω → 内部击穿，必换
❌ 阻值飘移：浸泡后阻值不变 → 玻璃珠裂了，必换

🔄 更换步骤

1
关机断电 + 冷透
热端必须冷到室温，否则拔玻璃珠会断在里面。KLP1 关机后等 15-20 分钟。
2
拔插头
从主控板或热端板拔下白色 2P 接头。压住卡扣再拔，别硬拉线。
3
拔出旧热敏电阻
用尖嘴钳夹住玻璃珠（只夹玻璃部分，别夹线），直直往外拔。
⚠️ 卡住拔不出：加热的料把玻璃珠糊在孔里。用电烙铁加热孔周围（200°C）让料软化，再拔。
4
清孔 + 装新
• 用 2.5mm 钻头 或 细砂纸 轻轻磨孔内壁，去掉碳化料
• 孔内挤一点 导热硅脂（信越 7762 或随意，2 元一管）
• 新的热敏电阻玻璃珠完全插入到底，不能留空隙
• 多余的线塞回线槽，别压到喷嘴或加热块

⚡ 加热电阻：万用表测试 + 更换

📋 测试方法

万用表拨到电阻档
拨到 200Ω 或 2KΩ 档。加热电阻的阻值是 11-15Ω，比较小。

📝 测试步骤

1
测冷态阻值
万用表表笔接加热电阻两脚（红/白线，无极性）。正常应该是 11-15Ω（24V/40W=14.4Ω）。
~13 Ω ✅
2
判断好坏
✅ 正常：11-15Ω
❌ 开路：OL → 内部电热丝烧断，必换
❌ 短路：0Ω 或几Ω → 击穿，主控会报加热异常，必换
❌ 外壳变形/烧焦：绝缘陶瓷坏了，必换（你拍的这颗就是这个情况）

🔄 更换步骤

1
冷透 + 拔插头
等热端完全冷到室温，拔掉加热电阻插头。
2
拧松卡箍/卡簧
不同机型固定方式不同：
• KLP1：两颗 M3 螺丝夹紧加热电阻
• 用 2.5mm 内六角拧松 1-2 圈即可
3
抽出旧 + 塞新
用尖嘴钳夹住加热电阻陶瓷块（别夹线）直直抽出。
新的加热电阻塞到底，重新拧紧卡箍螺丝（不要拧太紧，陶瓷会裂）。
⚠️ 安装加热电阻关键细节
• 塞到底：加热电阻头部必须贴合喷嘴座，否则升温慢、温度上不去
• 卡箍不要拧太紧：陶瓷是脆的，拧裂了会漏电
• 线不压：塞回线槽时别让线卡在喷嘴和加热块之间，否则高温烤焦

🔩 加热块 + 喷嘴：拆装

📋 工具

需要的工具
• 7mm 套筒扳手 或 棘轮扳手（拆喷嘴）
• 1.5mm / 2.5mm 内六角
• 尖嘴钳
• 旧的喷嘴钻头（0.4mm）或冷拔用耗材（PLA/PETG）

📝 拆装步骤

1
预热到 200°C（必须热拆）
喷嘴座和加热块是热配合，冷拆会卡死。
操作：
• 屏幕：MENU → Temperature → Hotend → 200°C
• 或 Klipper 终端：SET_HEATER_TEMPERATURE HEATER=extruder TARGET=200
• 等温度稳定到 200°C 再继续
2
拧出旧喷嘴
用 7mm 套筒 套在喷嘴上，逆时针拧出。
⚠️ 防烫：戴隔热手套！
⚠️ 方向：向下看 KLP1 喷嘴，逆时针松，顺时针紧（E3D 标准）
3
检查加热块状态
看加热块内孔是否有料堆积、碳化痕迹。用 0.4mm 钻头或针通一下（小心别断在里面）。
4
装新喷嘴
1. 保持 200°C 温度（让螺纹热胀）
2. 手动把新喷嘴顺时针拧到底（不要用工具，用手感觉到贴合）
3. 用套筒再拧 1/4 圈（约 90°），让密封圈压紧
4. 降到 0°C 后再拧 1/8 圈（冷紧一步，防止漏料）
5. 完全冷后复紧 1/8 圈
⚠️ 喷嘴更换最容易踩的坑
• 冷拧喷嘴：螺纹咬死，拧下来喷嘴就废了
• 拧太紧：加热块开裂（铝合金的）
• 忘了冷紧：热膨胀时漏料，碳化糊喷嘴
• 没预热装热敏电阻：孔里有料固化，热敏电阻插不进去

🔄 完整 8 步流程

1
关机断电 + 拔电源
断 KLP1 电源，等热端完全冷到室温（15-20 分钟）。
2
拆打印头外壳
KLP1 通常是 2-3 颗螺丝 固定打印头盖。拧下，露出热端。
3
拔所有插头
记录每个插头的位置（建议拍照），拔下：
• 加热电阻 2P/3P
• 热敏电阻 2P
• 风扇 2P（标记哪个是热端风扇）
• 探针（如果有）
4
拆旧加热块（预热到 200°C）
1. 开机 → 屏幕设置热端 200°C
2. 等到 200°C
3. 7mm 套筒逆时针拧出喷嘴
4. 松开喉管卡扣，抽出喉管
5. 松开加热块固定螺丝，取下加热块
5
清理料渣
• 加热块内孔：0.4mm 钻头轻轻通一下
• 喉管：用冷拔（把料挤到 240°C → 拔料 → 冷却后再拔）
• 检查风扇叶片是否被碳化料糊住
6
装新三件套
1. 加热电阻塞到加热块中间孔 → 卡箍轻拧紧
2. 热敏电阻孔挤一点点导热硅脂 → 玻璃珠完全插到底
3. 喉管从上方插入，穿过加热块到底（要看到喉管从喷嘴孔露头）
4. 保持 200°C：手动拧喷嘴顺时针到底 → 套筒再拧 1/4 圈
7
冷紧 + 检查
1. 关闭加热，让热端自然冷却到室温（不要风扇吹）
2. 完全冷后，用套筒再拧 1/8 圈（冷紧防漏）
3. 万用表复测：热敏电阻 100KΩ，加热电阻 13Ω
4. 检查线材没有压到发热部分
8
通电测试
1. 插好所有插头（参考第 3 步的照片）
2. 装回打印头外壳
3. 开机 → 温度应该能正常升到 200°C（60-90 秒）
4. Klipper 终端输入 QUERY_ADC 检查温度读数
5. 推一段料测试，确认不漏
💡 完整换装后建议
• 跑一次 PID_CALIBRATE HEATER=extruder TARGET=220 重做 PID
• 检查 printer.cfg 里 extruder 段配置：
  sensor_type: ATC Semitec 104GT-2（你 KLP1 的）
  heater_pin: PA0（加热电阻输出）
• 重新做 Z 偏移校准（热端位置变了，Z 会有偏差）
