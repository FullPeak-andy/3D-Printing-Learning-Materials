🔵 i3 直角坐标（Prusa MK4 / 拓竹 A1 / Ender 3）

i3 Animation · 拓竹 A1 风格
X 导轨（左右移动）
Y 导轨（前后移动）
Z 丝杆（上下移动）
工具头

🎬 运动原理

工具头在 X 导轨上左右滑 → X 导轨装在 Y 导轨上前后滑 → Y 导轨装在 Z 丝杆上上下滑

每个轴独立电机驱动（4 个电机：X、Y、Z、E），3 段运动完全解耦

🟢 CoreXY（Voron 2.4 / KLP1 / K1 Max / X1 Carbon）

CoreXY Animation · Voron 2.4 风格
Y 导轨 × 2（工具头前后移动）
X 皮带（X 型绕法，左右移动）
Z 丝杆
CoreXY 拉杆 × 2

🎬 运动原理

2 根 Y 导轨架在前后，工具头在 2 根 Y 导轨上前后滑

X 方向由左右 2 个电机 + 皮带 "X 型绕法" 驱动 → 工具头左右滑

Z 方向由 1 根丝杆驱动（和 i3 一样）

💡 X 和 Y 共用 2 个电机，所以电机数比 i3 少 1 个（5 个：2 个 XY + 1 个 Z + 1 个 E）

🟣 Delta 三角洲（FLSUN Super Racer / FLSUN V400 / ANYCUBIC Kossel）

Delta Animation · FLSUN Super Racer 风格
3 根垂直导轨（3 个角）
3 根碳纤维拉杆（顶部 3 角到中央）
中央打印头（被动吊挂）
圆形热床

🎬 运动原理

3 个滑车分别在 3 根垂直导轨上独立上下运动

3 根碳纤维拉杆从 3 个滑车同时拉中央打印头到 X/Y 位置

Z 方向 = 3 根滑车同步上抬/下放（无独立 Z 丝杆）

💡 工具头是被动吊挂的（被 3 根拉杆拉），运动件质量极轻
