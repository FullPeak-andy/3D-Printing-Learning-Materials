🖨️ 段 4：拿到 STL 后 → 切片 → 打印 流程

1
打开 STL 文件

用 OrcaSlicer（免费开源，Andy 哥首选）或 PrusaSlicer 打开任意 STL：

# OrcaSlicer 切片 ERCF_Main_Body.stl OrcaSlicer C:\3DPrint\ERCF_V3_full\ERCF_v3\STLs\ERCF_Main_Body.stl
2
设置切片参数（Voron 圈标准）

Voron 套件推荐参数（多色打印件也适用）：

参数推荐值说明
层高0.2 mm结构件标准
壁厚4 周壁（1.6 mm）结构强度
顶/底层5 层结构件
填充40% 陀螺 / 立方体平衡强度和材料
材料ABS / ASA / PC耐高温，Voron 件首选
温度240-260°CABS 245 / ASA 250 / PC 265
热床100-110°CABS / ASA 必须
封闭舱必装ABS 翘曲控制
支撑按需只对悬垂 > 45° 部位加
3
导出 G-code → 拷贝到打印机 SD 卡

切片完成后：切片 → 导出 G-code → 拷贝到 SD 卡 → 插入打印机打印

Andy 哥建议：用 KLP1 或 大鱼 SPA 打印 Voron 件（这俩 i3 留作"打印件工厂"），K1 改造机专攻最终打印

4
全套装机件清单（BOM 校验）

打印件完成后，参考每个仓库的 BOM_*.csv 文件采购机械件 + 电子件：

# 打开 BOM 文件查看零件清单 notepad C:\3DPrint\ERCF_V3_full\ERCF_v3\Docs\BOM_ERCF_V3.0.csv

典型 BOM 包括：电机（TMC2209 / TMC5160）、编码器、舵机、微动开关、LED、GT2 皮带、轴承、螺丝、磁铁、MMB 控制板
